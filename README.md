# ia-learning-labs

Laboratorios introductorios de inteligencia artificial, IBM Watson Studio, análisis de datos y Machine Learning.

Este repositorio reúne prácticas documentadas paso a paso relacionadas con herramientas de inteligencia artificial, plataformas cloud, carga de datos, preparación de proyectos y fundamentos de Machine Learning.

## Laboratorios disponibles

| Laboratorio                                                                                  | Descripción                                                                                                                                                               | Herramientas                                                            |
| -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| [Creación de proyecto en IBM Watson Studio y carga de dataset](ibm-watson-studio-risk-fraud) | Creación de un proyecto `Risk_Fraud` en IBM Watson Studio, aprovisionamiento de Cloud Object Storage, configuración de Watson Machine Learning y carga de un dataset CSV. | IBM Cloud, Watson Studio, Watson Machine Learning, Cloud Object Storage |

## Objetivo del repositorio

Documentar laboratorios prácticos de inteligencia artificial y análisis de datos de forma clara, ordenada y reproducible.

Estos laboratorios complementan mi formación técnica y fortalecen habilidades relacionadas con:

* Uso de plataformas cloud.
* Configuración de entornos de IA.
* Carga y validación de datasets.
* Documentación técnica.
* Fundamentos de Machine Learning.
* Preparación de proyectos para análisis de datos.

## Enfoque

Este repositorio documenta laboratorios prácticos de inteligencia artificial, análisis de datos y plataformas cloud, con énfasis en aprendizaje aplicado, organización técnica y documentación paso a paso.
